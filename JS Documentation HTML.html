Intro to JavaScript

JavaScript, abbreviated JS, is a derivative of coding language C. It contains the standard building blocks of coding languages.

Key elements include if/else statements, operators, and control structures.

Core Javascript has two sides:

The client-side supplies objects to control objects and its Document Object Model (DOM).

The server-side supplies relevant objects to running JavaScript on a server.

JavaScript vs. Java

JavaScript and Java are similar, but they have two key differences:

JavaScript does not subscribe to the strong type checking and static type that Java endorses. JavaScript has a prototype based object model in contrast to Java's class-based object model. In JavaScript, what is inherited can change depending on the object. This is called "dynamic inheritance". JavaScript's more lax style also allows for functions to be defined loosely without any dreary declarative statements.

JavaScript is effectively the "cool parent" of Java. They can execute similar tasks, but Java has much more rigid syntax that must be followed, while in JavaScript the syntax is loose and not explicit.

Data Types

JavaScript syntax is defined by ECMAScript. ECMAScript serves as a standard for the language specification for JS. It defines the seven denominations of data types as:

  • null (empty)
  • undefined
  • number ex/ 13
  • string ex/ Hello world
  • symbol
  • boolean ex/ true/false
  • object (contain values)
Hello world Statement

The "Hello world" statement is the standard launching point to a journey into programming. In JavaScript, you can print the statement with the following line:

function greetMe(yourName) { alert("Hello " + yourName); } greetMe("World");
